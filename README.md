# Landing
---
[A github page](https://ddadea.github.io/Numerical-Solutions/) that numerically solves well-known pendulum problems. Produced in a web environment for simplicity and accessibility.


### Single Pendulum
---
Pendulum simulations using Euler, RK4, Semi-Implicit Euler, and Velocity Verlet.


### Double Pendulum
---
Double pendulum chaos with multiple numerical methods.
